# RAG Retriever + Reranker

This project implements a Retrieval-Augmented Generation (RAG) pipeline with:
- BM25 keyword-based retriever
- Dense semantic retriever
- Reranking stage
- LLM inference via Groq API

## Demo
Hugging Face Space: <[link](https://huggingface.co/spaces/gdgsa/rag-cooking-qa)>

## Notes
- API keys are not stored; users provide their own keys.
- Free Hugging Face Spaces may experience cold start.
